# FileUploader. Laravel 11 + Vue3

## Установка
- Создать mysql базу данных
- Создать .env файл (скопировать шаблон из .env.example) и прописать данные для подключения к бд
- `composer install`
- `npm install`
- `php artisan storage:link`
- `php artisan migrate`

- `php artisan serve`
- `npm run dev`
